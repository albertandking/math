# 前言

## 为什么写这本书

机器学习已经进入了很多学习者的视野，但真正开始学习时，最常见的障碍往往并不是编程本身，而是数学表达带来的心理压力。许多初学者一看到公式，就会立刻觉得内容“太抽象”“太理论”或者“离应用太远”；也有一些读者虽然能够跟着代码运行模型，却始终不确定模型内部到底发生了什么。这种脱节会让后续学习越来越吃力，因为机器学习中的很多核心概念，无论是线性回归、梯度下降、概率模型还是神经网络，本质上都离不开数学语言的支撑。

这本教材的出发点，正是帮助这一类读者跨过最初的门槛。它不是为数学基础很强、希望迅速进入严格推导的读者准备的，而是为那些愿意认真理解机器学习、却暂时缺少系统数学训练的学习者准备的。全书会尽量把抽象概念拆开来讲，把公式、图像、表格、自然语言和 Python 小实验放在一起，让读者逐步建立一种更稳定的阅读能力：看到数学表达时，不再本能地退缩，而是能够一点点读懂它在描述什么关系、对应什么变化、解决什么问题。

## 这本书写给谁

本书主要面向以下几类读者：第一类是数学基础较弱、知识结构不完整的学习者，尤其是没有完整接受过中国大学数学体系训练的学生；第二类是已经接触过一些机器学习课程，但对公式、推导和模型机制仍然感到模糊的读者；第三类是希望从“会调用库函数”进一步走向“能够理解模型表达”的初学者。我们默认读者具备最基础的 Python 阅读能力，能够看懂简单变量、循环和函数调用，但不要求读者一开始就熟悉线性代数、微积分或概率统计的系统课程内容。

如果你希望阅读的是一套强调严格证明、完整公理体系和高强度推导训练的纯数学教材，那么这本书可能不是最合适的起点。它更重视理解顺序，更重视概念之间的连接，也更重视“机器学习里为什么要学这个数学”这一问题。

## 本书怎样组织内容

全书围绕“机器学习所需要的数学”展开，主线大致分为五段。第一段先解决公式阅读和符号理解问题，让读者建立最基本的数学阅读能力；第二段进入向量、矩阵和线性变换，帮助读者理解数据、参数和模型表达；第三段进入微积分与优化，理解导数、梯度和训练过程；第四段进入概率统计，帮助读者理解不确定性、分布和估计；第五段再回到具体模型，通过线性模型、降维方法和神经网络把前面的数学知识串联起来。

这样的安排并不是为了把数学知识“按学科完整讲一遍”，而是为了尽量让每一部分都服务于后面的模型理解。换句话说，本书不是先学完数学再碰机器学习，而是围绕机器学习真正需要的那部分数学去组织内容。

## 应该怎样使用这本书

阅读这本书时，最重要的原则不是“每个符号都一次记住”，而是“先理解它在表达什么关系”。因此，建议读者尽量按照章节顺序推进，尤其是在前四章阶段不要急于跳读。第 1 章和第 2 章的任务是建立阅读习惯与变化视角，第 3 章和第 4 章则是后续几乎所有模型表达的基础。只要这几个起步章节真正读顺了，后面遇到导数、梯度、概率和损失函数时，理解成本会明显下降。

每一章都会尽量按照类似的节奏展开：先说明本章为什么重要，再给出直觉解释，然后引入核心定义、例题与图示理解，最后再用 Python 小实验帮助读者把抽象概念变成可观察的对象。阅读时不必死守从头到尾一遍读完的方式。如果某个定义一开始不够顺，可以先去看图示理解；如果图示仍然不够直观，可以先跑一遍小实验，再回来看公式。对初学者来说，理解往往不是线性发生的，而是在文字、图像、例子和代码之间来回往返后逐步建立起来的。

## 图、公式和代码分别扮演什么角色

在这本教材里，图不是装饰，而是帮助读者建立直觉的桥梁；公式不是背诵材料，而是对关系的压缩表达；代码也不是为了展示技巧，而是为了把静态定义变成动态过程。因此，读者在阅读时最好不要把三者分开来看。看到图时，要主动问自己它在对应哪个关系；看到公式时，要主动问自己它是不是可以画出来、是不是可以举例说明；看到代码时，要主动观察变量如何变化、输出如何随输入改变。只有当图、公式和代码能够彼此指向同一个概念时，理解才会真正稳固。

## 对第一次阅读的建议

如果你是第一次系统学习机器学习数学，建议把目标定得务实一些。第一遍阅读时，不要强求自己把所有定义都背下来，也不必过早纠结形式化细节。更现实也更有效的目标是：能够读懂常见符号的含义，能够知道一个公式在描述什么关系，能够理解图像为什么这样变化，能够通过简单代码观察这种变化。等这些能力逐渐稳定之后，再回过头补更严格的定义和更深入的推导，效果通常会更好。

如果你在某一章里感到卡顿，也不意味着你“不适合学机器学习”。很多时候，困难并不是来自内容太难，而是来自知识之间缺少过渡。这本教材的工作，就是尽量把这些过渡补出来。

## 关于持续迭代

这本教材会持续改进。随着后续章节补全、插图完善和案例丰富，整本书的结构和表达也会继续打磨。如果你发现某个地方仍然不够清楚、不够像教材，或者某个概念还缺少一个更好的例子与图示，那么这些反馈都非常重要，因为它们恰好能帮助这本书更贴近真正的学习过程。

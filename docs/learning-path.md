# 学习路径

## 读者画像

这条学习路径主要面向以下学习者：

- 一看到符号和公式就容易紧张
- 学过一些中学数学，但使用起来并不熟练
- 听说过机器学习术语，却不理解其背后的数学机制

## 总体策略

这条路径不要求读者先完整学完一套形式化大学数学课程，而是按照机器学习真正需要的顺序，重建必要的数学基础。

## 阶段 0：先消除数学畏难

目标：

- 理解机器学习数学是可以分层学习的
- 不再把公式仅仅当作死记硬背的对象

重点内容：

- 变量
- 等式
- 函数
- 图像
- 基本代数变形

阶段结果：

- 能读懂简单公式，并解释一个变量变化时另一个量会如何变化

## 阶段 1：建立“量如何变化”的语言

目标：

- 理解函数就是输入到输出的对应规则
- 理解斜率和变化率的基本含义

重点内容：

- 一次函数
- 二次函数
- 指数与对数的直觉
- 局部变化与导数预告

机器学习联系：

- 模型输出依赖输入
- 损失会随着参数变化而变化

## 阶段 2：建立向量思维

目标：

- 从“单个数”过渡到“成组数字对象”

重点内容：

- 向量
- 距离
- 夹角
- 点积
- 矩阵乘法的直觉

机器学习联系：

- 每个样本都可以看作一个特征向量
- 很多预测本质上是加权求和

## 阶段 3：理解优化

目标：

- 理解模型是如何一步步变好的

重点内容：

- 导数
- 偏导数
- 梯度
- 梯度下降
- 学习率

机器学习联系：

- 训练过程就是让损失函数变小
- 梯度告诉我们参数该往哪个方向调整

## 阶段 4：理解不确定性

目标：

- 理解为什么数据会有噪声，为什么预测会存在不确定性

重点内容：

- 概率
- 条件概率
- 随机变量
- 期望
- 方差

机器学习联系：

- 分类模型常常输出概率
- 模型泛化离不开对不确定性的理解

## 阶段 5：理解数据摘要与统计思维

目标：

- 把具体数据样本和统计分析思维联系起来

重点内容：

- 均值
- 方差
- 协方差
- 估计
- 过拟合的直觉

机器学习联系：

- 数据的统计性质会影响模型行为
- 训练集和测试集指标都带有统计波动含义

## 阶段 6：回到完整模型

目标：

- 把前面所有数学知识整合到具体模型中

重点内容：

- 线性回归
- 逻辑回归
- PCA
- 浅层神经网络

阶段结果：

- 能识别一个模型使用了哪些数学工具
- 能从高层次解释模型训练过程

## 建议学习节奏

- 一次只读一个较短小节
- 每学完一点就立即做几道很小的练习
- 手动画图，重新画函数图像和几何示意
- 建一个自己的“符号笔记本”
- 学完一个模型后，再回头复习前面的数学章节

## 前期应弱化的内容

为了保护学习动机，以下内容不应在前期成为主线：

- 过强的极限严格化证明
- 很长的定理证明链条
- 过于抽象的向量空间形式主义
- 测度论概率

## 成功标准

对这个阶段的学习者来说，不需要先把自己训练成数学专业学生。

更现实也更重要的目标是达到下面这个状态：

“我知道这个公式在描述什么，为什么它会出现在机器学习里，以及这些量之间是如何一起变化的。”

# 教材总纲

## 教材定位

本教材聚焦于“机器学习所需要的数学”，强调可理解性、可进入性和与应用场景的连接。

目标读者主要包括：

- 数学基础薄弱或知识零散的学习者
- 没有完整修过中国大学数学体系课程的学生
- 希望真正理解机器学习模型，而不仅仅是调用库函数的初学者

## 教学理念

### 1. 先回答“为什么学”，再进入“怎么做”

读者应先理解一个数学概念在机器学习中为什么重要，再学习它是什么，最后再看公式如何表达。

### 2. 优先建立几何直觉与数值直觉

在可能的情况下，概念引入尽量依靠以下方式展开：

- 图像和几何示意
- 简单数值例子
- 数据问题故事
- 与回归、分类、降维等机器学习任务的联系

### 3. 控制证明强度

本教材不是纯数学教材。证明应当有选择地出现，只有在确实能帮助理解时才展开。

### 4. 明确区分学习深度

每一节建议将内容区分为：

- `核心`：大多数机器学习学习者必须掌握
- `拓展`：重要但第一次阅读可以略读
- `进阶`：更深入的数学内容，可选学习

## 学习目标

完成本教材后，读者应能够：

- 不再惧怕常见机器学习公式
- 理解向量、矩阵、导数、梯度、概率、期望等核心概念
- 理解损失函数和优化过程在做什么
- 从数学层面把握线性回归、逻辑回归、PCA 和神经网络的基础机制

## 全书结构

### 第一部分：数学准备

1. `01-foundations.md`
   - 数学符号、变量、等式、函数、图像
   - 如何阅读公式
   - 常用代数基础回顾

2. `02-functions-and-change.md`
   - 函数视角
   - 斜率、变化率与局部近似
   - 为什么导数重要

### 第二部分：面向数据的线性代数

3. `03-vectors-and-geometry.md`
   - 向量作为点、箭头和特征容器
   - 范数、距离、夹角、点积
   - 这些概念在机器学习中的几何意义

4. `04-matrices-and-transformations.md`
   - 矩阵作为数据表与线性映射
   - 矩阵乘法的直觉解释
   - 线性方程组

5. `05-eigenvalues-and-svd.md`
   - 特征值与特征向量
   - 奇异值分解
   - PCA 的直觉基础

### 第三部分：面向学习过程的微积分

6. `06-single-variable-calculus.md`
   - 导数的直觉
   - 极值问题
   - 凸性的初步认识

7. `07-multivariable-calculus.md`
   - 偏导数
   - 梯度
   - 链式法则
   - 等高线与曲面直觉

8. `08-optimization.md`
   - 目标函数
   - 梯度下降
   - 学习率
   - 局部最优与全局最优

### 第四部分：概率统计基础

9. `09-probability-basics.md`
   - 事件、条件概率、独立性
   - 贝叶斯直觉

10. `10-random-variables-and-distributions.md`
    - 离散与连续随机变量
    - 期望与方差
    - 高斯分布、伯努利分布、二项分布

11. `11-statistics-and-inference.md`
    - 总体与样本
    - 估计的基本思想
    - 偏差与方差的直觉
    - 训练集、验证集、测试集的统计含义

### 第五部分：机器学习中的数学整合

12. `12-linear-models.md`
    - 最小二乘
    - 逻辑回归
    - 正则化直觉

13. `13-neural-network-math.md`
    - 仿射变换
    - 激活函数
    - 反向传播的基本直觉

14. `14-math-review-through-models.md`
    - 通过模型案例串联前面所有数学内容
    - 回归、分类、PCA 与浅层神经网络

### 第六部分：从基础走向进阶

15. `15-matrix-calculus-and-autodiff.md`
    - 标量、向量、矩阵之间的求导对象
    - 计算图、形状匹配与梯度组织
    - 自动求导与反向模式的基本机制

## 章节模板 { #chapter-template }

本节是全书章节结构的**唯一权威来源**。其他页面（如章节目录）若需说明章节结构，应链接到本节，而不要另抄一份，以免出现新旧不一致。

每一章原则上按如下结构展开（与当前正文章节统一结构保持一致）：

1. 本章导读
2. 学习目标
3. 本章为什么重要
4. 先修知识清单
5. 直觉解释
6. 核心概念
7. 例题与推导
8. 图示理解
9. Python 小实验
10. 与机器学习的联系
11. 常见误区
12. 练习题
13. 本章知识结构
14. 本章小结

## 写作规范

- 避免在一个小节中一次性引入过多新符号
- 同一节内尽量保持符号一致
- 为关键记号设置“符号说明盒”
- 段落尽量简短
- 公式出现后立刻解释其机器学习含义
- 明确指出哪些内容第一次阅读可以跳过

# 附录 A 术语、符号、读法与名词对照

本附录用于集中整理本书中会反复出现的数学符号、阅读约定、常见读法、机器学习相关术语以及全书的名词对照。对初学者来说，很多困难并不是来自某一个单独的概念，而是来自符号太多、名字太多、表达形式太多。本页的目标，就是把这些经常出现的记号先稳定下来，便于在阅读各章时随时回查。

## A.1 阅读约定

全书尽量遵循以下约定：

- 行内公式统一写作 `\(...\)`，独立公式统一写作 `\[...\]`
- 小写斜体字母通常表示某个数学对象，例如 `\(x\)`、`\(y\)`、`\(w\)`；它有时表示单个量，也有时在机器学习语境中直接表示一条样本、一个输入向量或一组参数，需要结合上下文判断
- 如果正文需要特别强调“这是一个向量或矩阵”，通常会通过坐标列表、列向量写法、矩阵写法或上下文说明来提示，而不强制全书每次都改用粗体字母
- 大写字母通常表示矩阵、数据表或函数整体，例如 `\(X\)`、`\(W\)`、`\(f\)`
- 同一个符号在同一小节内尽量保持单一含义，避免来回切换

这意味着：当你在正文里看到 `\(x\)` 或 `\(w\)` 时，不要先急着问“它是不是一定只能表示一个数字”，而应先看上下文是在讨论单个数、一个按顺序排列的对象，还是一批样本的组织方式。对初学者来说，这种“先看角色、再看记号形式”的阅读顺序，通常比死记一套排版规则更稳妥。

初学者在阅读公式时，不必急于“翻译成标准数学语言”，可以先做三件事：先看每个字母代表什么量，再看这些量之间通过什么运算连接，最后再问这个式子是在描述输入输出关系、几何关系，还是概率关系。

## A.2 常见运算符号怎么读

对初学者来说，先把符号读顺，往往比先把公式算对更重要。很多公式一眼看上去难，并不是因为概念本身太深，而是因为读者还没有形成稳定的“读法节奏”。下面这张表优先解决这个问题。

| 符号 | 常见读法 | 在本书中的常见理解 |
|---|---|---|
| \(=\) | 等于 | 左边和右边表示的是同一个结果或同一种关系 |
| \(+\) | 加 | 把几个量合并起来 |
| \(-\) | 减 | 表示差值、减少量，或负号 |
| \(\times\) | 乘 | 两个量相乘 |
| \(\cdot\) | 点乘、乘 | 在一般表达里常读作“乘”，在向量里常读作“点积” |
| \(/\) | 除以 | 表示一个量与另一个量的比值 |
| \(\frac{a}{b}\) | \(a\) 比 \(b\)、\(a\) 除以 \(b\) | 常表示比例、变化率或平均值 |
| \(^2\) | 平方 | 表示一个量乘以它自己 |
| \(^3\) | 立方 | 表示一个量连乘三次 |
| \(x^n\) | \(x\) 的 \(n\) 次方 | 指幂次关系 |
| \(\sqrt{x}\) | 根号 \(x\)、\(x\) 的平方根 | 常表示长度、标准差等量 |
| \((\ )\) | 括号 | 提醒这一部分先作为整体来看 |
| \([\ ]\) | 方括号 | 常用于期望、向量、区间或整体结构 |
| \(\{\ \}\) | 花括号 | 常表示集合或分类情况 |
| \(\sum\) | 求和、西格玛求和 | 把一串相似结构加起来 |
| \(\prod\) | 连乘、乘积 | 把一串相似结构乘起来 |
| \(|x|\) | \(x\) 的绝对值 | 表示数离 0 的距离 |
| \(\|\mathbf{x}\|\) | 向量 \(x\) 的范数、向量 \(x\) 的长度 | 表示向量有多大 |
| \(\cdots\) | 省略号 | 中间还有同类项没有展开 |
| \(\in\) | 属于 | 某个元素属于某个集合 |
| \(\to\) | 趋向于、映到 | 常用于极限、映射或“变化到” |
| \(\mid\) | 在……条件下 | 在条件概率里尤其常见 |
| \(\approx\) | 约等于 | 表示近似，不是完全相等 |
| \(\leq\) | 小于等于 | 表示不超过某个值 |
| \(\geq\) | 大于等于 | 表示不低于某个值 |

## A.3 全书高频希腊字母怎么读

机器学习数学里经常出现希腊字母。它们看上去陌生，但本质上只是另一套“字母名字”。先把它们读顺，再去理解它们在不同章节里具体表示什么，阅读压力会小很多。

| 符号 | 常见读法 | 中文近似读音 | 在本书中的常见角色 |
|---|---|---|---|
| \(\alpha\) | alpha | 阿尔法 | 系数、权重或某个参数 |
| \(\beta\) | beta | 贝塔 | 参数、系数 |
| \(\theta\) | theta | 西塔 | 模型参数的统称 |
| \(\lambda\) | lambda | 拉姆达 | 特征值、正则化强度 |
| \(\sigma\) | sigma | 西格玛 | 标准差、sigmoid 记号 |
| \(\mu\) | mu | 缪 | 均值、中心位置 |
| \(\eta\) | eta | 艾塔 | 学习率 |
| \(\pi\) | pi | 派 | 圆周率，或某些概率记号 |
| \(\phi\) | phi | 菲 | 函数记号、特征映射等 |
| \(\rho\) | rho | 柔 | 相关系数或某种比例参数 |
| \(\delta\) | delta | 德尔塔 | 变化量、误差量 |
| \(\Delta\) | Delta | 大德尔塔 | 总变化量 |
| \(\epsilon\) | epsilon | 艾普西隆 | 很小的量、误差项 |
| \(\nabla\) | nabla | 纳布拉 | 梯度符号 |

## A.4 常见数学记号

| 符号 | 常见含义 | 阅读提示 |
|---|---|---|
| \(x\) | 输入、自变量、某个特征值，或一条样本的输入向量 | 在简单函数里常表示单个输入；进入机器学习语境后，也常直接表示一条样本或输入对象 |
| \(y\) | 输出、结果、标签 | 常表示模型希望预测的量 |
| \(f(x)\) | 函数在输入 \(x\) 处的输出 | 可理解为“把 \(x\) 放进规则后得到的结果” |
| \(w\) | 参数、权重，或权重向量 | 在线性模型里既可能表示单个权重，也可能表示整组权重 |
| \(b\) | 偏置、截距 | 常表示整体平移量 |
| \(\mathbf{x}\) | 向量的常见写法 | 有些教材或英文资料会用粗体强调向量；本书正文不强制统一采用这种排版 |
| \(\mathbf{w}\) | 权重向量的常见写法 | 有些资料会用粗体强调“这是一整组权重”；本书正文更常直接结合上下文说明 |
| \(X\) | 数据矩阵、设计矩阵 | 可理解为“把很多样本排成一张表” |
| \(W\) | 参数矩阵、变换矩阵 | 常用于多维线性变换或神经网络层 |
| \(x_i\) | 第 \(i\) 个元素 | 下标用于区分位置或编号 |
| \(a_{ij}\) | 矩阵第 \(i\) 行第 \(j\) 列元素 | 读矩阵时先看行列位置 |
| \(\sum\) | 求和符号 | 表示把一组量加起来 |
| \(\prod\) | 连乘符号 | 表示把一组量乘起来 |
| \(\sqrt{x}\) | 平方根 | 表示“哪个数平方后等于 \(x\)” |
| \(|x|\) | 绝对值 | 表示数值离 0 的距离 |
| \(\| \mathbf{x} \|\) | 向量长度、范数 | 常用来衡量向量大小 |
| \(\mathbf{x} \cdot \mathbf{w}\) | 点积 | 可理解为“加权求和”或“方向匹配程度” |
| \(A^\top\) | 转置 | 表示把矩阵的行和列交换 |
| \(A^{-1}\) | 逆矩阵 | 只对可逆矩阵有定义 |
| \(\lambda\) | 特征值、正则化强度等 | 需结合上下文判断具体含义 |
| \(\sigma\) | 标准差、激活函数等 | 需注意它可能在不同章节含义不同 |
| \(\mu\) | 均值、期望附近的中心位置 | 概率与统计章节中常见 |
| \(\mathbb{E}[X]\) | 期望 | 可理解为随机变量的长期平均水平 |
| \(\mathrm{Var}(X)\) | 方差 | 衡量随机变量波动大小 |
| \(P(A)\) | 事件 \(A\) 的概率 | 表示某件事发生的可能性 |
| \(P(A \mid B)\) | 条件概率 | 表示“在 \(B\) 已经发生的条件下，\(A\) 的概率” |
| \(\nabla f\) | 梯度 | 表示多元函数变化最快的方向 |
| \(\frac{df}{dx}\) | 导数 | 表示函数相对于某个变量的变化率 |
| \(\frac{\partial f}{\partial x}\) | 偏导数 | 用于多元函数中只对一个变量求变化率 |
| \(L\) | 损失函数 | 用来衡量模型预测得好不好 |
| \(\eta\) | 学习率 | 控制每次参数更新走多大一步 |

## A.5 机器学习中的高频表达

| 表达 | 常见解释 | 在本书中的理解方式 |
|---|---|---|
| \(y = wx + b\) | 一元线性关系 | 先读成“输入变化时，输出按某个规则变化” |
| \(w \cdot x + b\) | 线性模型打分公式 | 先读成“把多个特征加权汇总，再整体平移” |
| \(Xw\) | 矩阵乘向量 | 先读成“把一批样本同时按同一个规则处理” |
| \(f(x; \theta)\) | 带参数的函数 | 先读成“函数形状由参数 \(\theta\) 决定” |
| \(\hat{y}\) | 预测值 | 帽子常表示“估计出来的量” |
| \(L(y, \hat{y})\) | 损失函数 | 比较真实值和预测值之间差距 |
| \(\arg\min\) | 使目标最小的取值 | 可读作“让它最小的那个参数是什么” |
| \(\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t)\) | 梯度下降更新 | 读成“沿着损失下降最快的反方向走一步” |

## A.6 全书名词统一约定

为了减少初学者在阅读时的负担，全书正文尽量遵循下面这些主用词约定。这里所说的“统一”，并不是把本来有差别的词强行写成同一个词，而是尽量先固定一个主用词，再把相关词之间的层级关系说明清楚。

| 主用词 | 相关词 | 本书中的统一方式 |
|---|---|---|
| 参数 | 权重、偏置、系数 | **参数**是总称；在线性模型和神经网络里，**权重**与**偏置**是参数的具体类型；当只讨论单个公式中的乘法常数时，也会出现**系数**这个词 |
| 输入 | 自变量、特征 | 在函数章节中保留**自变量**；进入机器学习语境后，默认优先写**输入**或**输入特征**；**特征**强调的是输入中的某个描述维度 |
| 输出 | 函数值、预测值、标签、目标值 | **输出**是最一般的说法；**函数值**用于纯数学语境；**预测值**用于模型给出的结果；**标签**或**目标值**用于真实答案，不与预测值混用 |
| 损失函数 | 损失、误差、残差 | **损失函数**是完整术语，正文可简称**损失**；**误差**是更一般的偏差概念；**残差**通常特指预测值与真实值之差，不与损失混用 |
| 样本 | 样本点、数据点 | 正文默认优先使用**样本**；需要强调它在空间中的位置时，可以临时写**样本点**；**数据点**不作为主用词 |
| 概率 | 可能性 | 解释直觉时可以说**可能性**，但进入正式表述后默认写**概率** |
| 激活函数 | 激活 | 正文在正式定义时使用**激活函数**，后文在上下文明确时可简称**激活** |
| 训练误差与验证误差 | 泛化误差 | **训练误差**和**验证误差**用于具体曲线或数据集表现；**泛化误差**强调模型对未见数据的整体能力，不与前两者直接混同 |

为了帮助读者后续阅读英文资料或框架文档，本书对一部分高频术语采取“**第一次出现时给出英文括注，后文以中文为主**”的策略。例如：向量（vector）、矩阵（matrix）、张量（tensor）、梯度（gradient）、随机变量（random variable）、分布（distribution）、损失函数（loss function）、激活函数（activation function）等。这样既能保持正文的中文阅读连续性，也能帮助读者把教材语言和工程语境接起来。

## A.8 怎样更稳地读公式

如果你一看到公式就紧张，可以先不要把它当作“需要立刻计算的东西”，而把它当作一句被压缩过的话。比较稳妥的阅读顺序通常是这样的：先识别等号两边谁是输入、谁是输出；再看有没有加法、乘法、求和、平方这类熟悉运算；然后问自己这个式子更像是在描述规则、几何关系还是概率关系；最后再结合图示或代码观察它是否真的表现出你读到的那种变化。很多时候，公式之所以难，并不是因为它本身特别深，而是因为还没有找到合适的阅读节奏。

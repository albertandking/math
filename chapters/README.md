# 章节目录

本目录用于存放教材正文。

## 编写原则

- 每章都要先解释“为什么要学这一章”
- 每章至少提供一个生活化例子和一个机器学习例子
- 对抽象概念优先配图，再进入公式
- 对关键概念尽量提供一段可执行的 Python 小实验
- 练习题要从“读懂概念”开始，而不是直接进行复杂计算

1. `01-foundations.md`：数学准备与公式阅读
2. `02-functions-and-change.md`：函数、图像与变化率
3. `03-vectors-and-geometry.md`：向量与几何直觉

6. `06-single-variable-calculus.md`：一元微积分基础
7. `07-multivariable-calculus.md`：多元微积分与梯度
8. `08-optimization.md`：优化方法
9. `09-probability-basics.md`：概率论基础
10. `10-random-variables-and-distributions.md`：随机变量与常见分布
11. `11-statistics-and-inference.md`：统计学基础与估计

## 建议编写顺序

建议优先扩写以下章节：

- 第 1 章：帮助读者建立符号和公式的阅读能力
- 第 2 章：建立函数与变化的视角
- 第 3 章：建立向量思维
- 第 4 章：建立矩阵与线性映射的直觉
- 第 6 章：建立导数与优化基础
- 第 9 章：建立概率基础

## 每章统一模板

每章原则上至少包含以下部分：

1. 本章为什么重要
2. 先修知识清单
3. 直觉解释
4. 核心定义
5. 例题与推导
6. 图示理解
7. Python 小实验
8. 配图建议
9. 与机器学习的联系
10. 常见误区
11. 练习题
12. 本章小结

## 阅读方式建议

- 零基础读者：按第 1 -> 2 -> 3 -> 4 章顺序推进
- 有基础读者：可从向量、矩阵或概率章节跳读
- 每章都尽量按“直觉 -> 图示 -> Python 小实验 -> 公式 -> 模型联系”的顺序阅读

## 每章统一模板

每章原则上至少包含以下部分：

1. 本章为什么重要
2. 先修知识清单
3. 直觉解释
4. 核心定义
5. 例题与推导
6. 图示理解
7. Python 小实验
8. 配图建议
9. 与机器学习的联系
10. 常见误区
11. 练习题
12. 本章小结

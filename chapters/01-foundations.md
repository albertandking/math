# 第 1 章 数学准备与公式阅读

## 本章为什么重要

很多学生并不是完全学不会机器学习中的数学，而是卡在更前面的一步：看不懂公式到底在说什么。

例如，看到下面这样的式子时：

```text
y = wx + b
```

不少初学者会同时产生几个问题：

- `y` 是什么
- `x` 是什么
- `w` 和 `b` 为什么也是字母
- 这个式子到底是在计算、描述规律，还是在定义一个模型

因此，本章不是要一下子讲很多高等数学内容，而是先帮助读者建立最基本的阅读能力：

- 看懂变量和符号
- 看懂公式是在描述什么关系
- 看懂函数和图像之间的对应
- 看懂“一个量变化时，另一个量如何变化”

这些能力看起来基础，但它们是后续学习线性代数、微积分、概率统计和机器学习模型的入口。

## 先修知识清单

阅读本章前，读者最好已经接触过以下内容：

- 加减乘除
- 简单代数式
- 一元一次方程
- 平面直角坐标系的最基础概念

如果这些内容已经比较模糊，也不必担心，本章会尽量通过例子把它们重新唤醒。

## 直觉解释

### 1. 数学公式本质上是在压缩表达

一句自然语言：

“学生每天学习时间越多，测试成绩通常越高。”

可以被压缩成一种更数学化的表达：

```text
成绩 = 某个关于学习时间的函数
```

如果进一步假设这个关系近似线性，那么可以写成：

```text
成绩 = a × 学习时间 + b
```

也就是：

```text
y = ax + b
```

这里的公式并不是为了吓人，而是为了把“量和量之间的关系”写得更短、更清楚、更便于计算。

### 2. 字母不是障碍，而是“可变化量”的名字

初学者常把数字看作真实，字母看作抽象而可怕。实际上，字母只是一个名字。

例如：

- `x` 可以表示学习时间
- `y` 可以表示考试成绩
- `w` 可以表示某个特征的重要程度
- `b` 可以表示基础水平或偏移量

在机器学习里，字母会更多，因为我们同时要描述：

- 输入
- 输出
- 参数
- 误差
- 损失函数

所以本章最关键的适应不是“会算”，而是“接受用字母表达关系”。

### 3. 图像是理解公式的桥梁

对基础薄弱的读者来说，只看公式往往太抽象；只看文字又不够精确。图像正好起到中介作用。

如果一个函数写成：

```text
y = 2x + 1
```

那么它表达的是：

- `x` 每增加 1
- `y` 就增加 2
- 即使 `x = 0`，`y` 也不是 0，而是 1

这些变化关系一旦画在平面坐标系里，就会变成一条直线。很多看似抽象的数学内容，一旦画出来，就会突然变得能理解。

## 核心概念

### 1. 变量

变量是可以取不同数值的量。

例如：

- 一个学生每天学习的小时数可以变化
- 一套房子的面积可以变化
- 一条样本记录中的年龄可以变化

因此，这些都可以看成变量。

在机器学习中，变量经常对应数据中的一个特征，或者模型中的一个参数。

### 2. 常量

常量是在某个表达中固定不变的量。

例如在：

```text
y = 3x + 2
```

中：

- `x` 和 `y` 可以变化
- `3` 和 `2` 是常量

不过要注意，是否是常量，取决于当前讨论的对象。在一个场景里固定的量，在另一个场景里也可能成为变量。

### 3. 等式

等式表示左右两边相等。

例如：

```text
2 + 3 = 5
```

或者：

```text
y = 2x + 1
```

第二个例子并不是说 `y` 永远等于一个固定数字，而是说：只要 `x` 取了某个值，`y` 就按照右边的规则被确定下来。

### 4. 函数

函数可以理解为一种规则：

- 给定输入
- 经过某种确定关系
- 得到输出

例如：

```text
f(x) = 2x + 1
```

它表示一个名叫 `f` 的规则。输入 `x` 后，输出为 `2x + 1`。

如果：

- `x = 1`，那么 `f(1) = 3`
- `x = 2`，那么 `f(2) = 5`
- `x = 5`，那么 `f(5) = 11`

这就是函数最朴素的含义。

## 例题与推导

### 例 1：如何读懂 `y = 2x + 1`

这不是一句需要死记的“公式”，而是一条变化规则。

可以按下面的顺序读：

1. `x` 是输入
2. 把输入乘以 2
3. 再加上 1
4. 得到输出 `y`

取几个具体数值：

| `x` | `y = 2x + 1` |
| --- | --- |
| 0 | 1 |
| 1 | 3 |
| 2 | 5 |
| 3 | 7 |

从这个表中可以看出：

- `x` 每增加 1，`y` 增加 2
- 这是一种稳定的线性变化

### 例 2：用函数描述学习时间与成绩

假设一个非常理想化的模型：

```text
成绩 = 5 × 学习小时数 + 50
```

如果用字母表示：

```text
y = 5x + 50
```

这里可以解释为：

- `x`：一天学习多少小时
- `y`：预测成绩
- `5`：每多学 1 小时，成绩预计增加 5 分
- `50`：即使学习时间记为 0，基础成绩仍有 50 分

这就是最初级的“数学建模”。

虽然真实世界远比这复杂，但机器学习模型的核心思想也正是：用数学关系描述输入和输出之间的联系。

### 例 3：为什么 `f(x)` 和 `y` 经常一起出现

有时你会看到：

```text
y = f(x)
```

这表示：

- `f` 是函数名
- `x` 是输入
- `f(x)` 是函数在输入 `x` 时的输出
- `y` 用来表示这个输出值

所以：

```text
y = f(x)
```

可以理解成：

“输出 `y` 由函数 `f` 作用在输入 `x` 上得到。”

## 配图建议

本章建议至少配三张图：

1. 变量与常量示意图  
   用一个简单式子标出哪些量能变、哪些量不变。

2. 函数表格到图像的转化图  
   先给出 `x` 与 `y` 的取值表，再把这些点画到坐标系中，帮助学生理解“公式、表格、图像”其实是在表达同一件事。

3. 一次函数 `y = 2x + 1` 的直线图  
   在图上标清截距和“每向右 1 格，向上 2 格”的变化方式。

## 与机器学习的联系

这一章虽然基础，但和机器学习直接相关。

### 1. 机器学习模型本质上也是函数

最简单的理解是：

- 输入一些特征
- 模型输出一个预测结果

这本质上就是函数。

例如：

```text
房价 = 某个关于面积、地段、楼龄的函数
```

或者：

```text
是否点击广告 = 某个关于用户特征的函数
```

### 2. 参数决定函数形状

在式子：

```text
y = wx + b
```

中：

- `x` 是输入
- `y` 是输出
- `w` 和 `b` 是参数

参数变化，整条函数关系就会变化。后面学习线性回归和神经网络时，读者会不断看到这种结构。

### 3. 后面的复杂公式，其实是这个思路的扩展

以后你会看到更复杂的表达，例如：

```text
z = w_1x_1 + w_2x_2 + \cdots + w_nx_n + b
```

它看起来比 `y = 2x + 1` 复杂很多，但本质思路没有变：

- 多个输入
- 若干权重
- 一个输出

所以只要本章的阅读能力建立好了，后面的复杂公式就不再是完全陌生的符号堆。

## 常见误区

### 误区 1：字母越多，数学越高级

不一定。很多时候字母多，只是因为需要同时表示多个量，并不意味着概念本身更难。

### 误区 2：公式是拿来背的

更准确地说，公式是拿来表达关系的。先理解关系，再记公式，效果会好得多。

### 误区 3：图像只是辅助，不重要

对初学者而言，图像经常是理解函数、变化率和优化过程的关键入口。

### 误区 4：机器学习公式和基础数学没关系

恰恰相反。复杂模型通常只是把基础数学对象组合起来。变量、函数、图像、向量、概率，这些基础内容都是机器学习公式的“零件”。

## 练习题

1. 在表达式 `y = 4x + 3` 中，指出哪些是变量，哪些是常量。
2. 计算当 `x = 0, 1, 2, 3` 时，`y = 3x + 2` 的取值。
3. 用自然语言解释公式 `y = 5x + 10` 表示什么样的变化关系。
4. 设 `f(x) = x + 7`，求 `f(1)`、`f(5)` 和 `f(10)`。
5. 请你自己举一个生活中的例子，尝试写成“输出 = 某个关于输入的函数”的形式。

## 本章小结

本章的重点不是计算技巧，而是数学阅读能力。

学完本章后，读者应当逐步能够：

- 接受用字母表示量
- 理解公式是在描述变量之间的关系
- 理解函数就是输入到输出的规则
- 在公式、表格和图像之间建立联系
- 初步理解机器学习模型也可以看作函数

如果这一章掌握得比较扎实，后面进入向量、导数、概率和优化时，理解难度会明显下降。

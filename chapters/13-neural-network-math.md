## 本章导读

第 12 章里，我们已经看到线性模型如何把特征做加权求和，再通过损失函数和梯度学习参数。但如果模型始终只是单层线性结构，那么它的表达能力就会受到很大限制。现实中的很多关系并不是简单直线或单个超平面能够描述的，例如图像识别、语音理解和复杂文本分类，都需要更强的表示能力。也正因为如此，神经网络就会自然出现。

很多初学者第一次接触神经网络时，会觉得它是一个完全不同的新世界，似乎前面的向量、矩阵、导数和概率都突然不够用了。实际上，神经网络并没有推翻前面的数学基础，而是在系统地重复和组合它们。每一层本质上仍然在做仿射变换 \(Wx+b\)，只是中间再接一个非线性激活函数；整张网络本质上仍然是一个复合函数；而参数更新所依赖的梯度，依然来自链式法则和梯度下降。

因此，本章的重点不是把神经网络包装成神秘黑箱，而是帮助读者看到它背后的连续性。只要你能够看清三层结构，神经网络就会自然很多：第一层是线性组合，第二层是激活带来的非线性，第三层是损失对所有参数的反向梯度传递。只要这三层真正建立起来，后面学习更深网络时就不会那么容易失去方向。

## 学习目标

学完本章后，读者应当能够达到以下要求：

- 能够用“多层仿射变换 + 激活”解释神经网络结构
- 能够理解激活函数为什么让模型突破线性表达限制
- 能够解释反向传播为什么本质上依赖链式法则
- 能够把神经网络训练与前面章节的向量、矩阵、梯度和优化方法联系起来

第一次阅读本章时，不必急于掌握所有网络变种或复杂实现细节。更重要的是先真正看清：神经网络并没有离开前面学过的数学，而是在把这些基础对象大规模组合起来。只要这层连续性感建立起来，后面遇到更深更复杂的模型时就不会那么容易觉得跳脱。

## 本章为什么重要

神经网络之所以重要，不只是因为它在现代机器学习应用中极为常见，更因为它把前面几乎整本书学过的数学真正集中到了一起。输入样本需要用向量表示，层与层之间的计算需要用矩阵乘法表达，激活函数需要依赖函数与图像直觉，损失函数与参数更新需要依靠导数、梯度和优化方法，而反向传播更直接地把链式法则变成了一种系统算法。换句话说，神经网络不是一个附加专题，而是全书数学主线的一次综合应用。

从表达能力来看，神经网络的重要性还体现在“非线性”。单层线性模型再强，本质上仍然只能形成线性决策边界或线性回归表达。现实世界中的很多规律显然更加复杂，这时就必须依靠激活函数引入非线性，让模型能够表示更丰富的映射关系。也正因为如此，理解神经网络时最关键的不是“层数越多越高级”，而是先理解为什么非线性会成为能力提升的核心。

最后，反向传播的重要性体现在训练机制上。模型越复杂，参数越多，就越需要一种有效方式来计算“每个参数应如何更新”。如果没有链式法则和反向传播，深层网络几乎无法实际训练。因此，本章不仅在讲网络结构，也在讲现代机器学习训练为何能够真正运转起来。

## 直觉解释

### 1. 单个神经元本质上还是加权求和

如果把一个神经元单独拎出来，它做的第一件事并不复杂，仍然是把输入向量按权重做加权求和，再加上偏置：

\[
z = w^\top x + b
\]

也就是说，神经网络并不是从零发明了新的基础计算，而是在延续线性模型的核心结构。对初学者来说，这一点非常重要，因为它能打消“神经网络完全脱离前面内容”的错觉。

### 2. 激活函数让模型不再只会画直线

如果网络里所有层都只做线性变换，而没有任何非线性激活，那么即使堆很多层，整体上仍然可以压缩成一个更大的线性变换。这意味着模型表达能力并没有真正突破线性范围。激活函数的作用，正是在这里引入关键改变。它让每一层输出不再只是线性叠加，从而使整个网络能够表达更复杂的模式。

从图像角度看，这件事可以理解成：模型不再只能画一条直线，而开始能够拼接和弯折出更复杂的决策形状。这也是为什么神经网络能比简单线性模型表达更复杂关系的根本原因。

### 3. 多层网络是在逐步变换表示

如果把一层看成“把原始输入重新组合成新的特征表示”，那么多层网络就可以理解为：第一层先做一次表示变换，第二层再在前一层表示的基础上继续变换，后面各层不断重复这一过程。这样一来，网络不只是“算结果”，而是在逐层构造更适合当前任务的特征表达。

这层理解对机器学习学习者特别重要，因为它说明深度网络的优势并不只来自“参数更多”，而来自“能逐层形成更适合问题的表示”。

### 4. 反向传播是在把误差信息一层层传回去

网络前向计算时，信息从输入流向输出；而训练时，损失对参数的影响则必须从输出往回传。反向传播做的正是这件事：利用链式法则，把损失函数对后层输出的变化敏感度，逐层传回到更前面的参数上。这个过程听起来复杂，但其本质仍然和第 7 章的链式法则完全一致，只是现在变量更多、层次更深而已。

## 核心概念

### 1. 仿射变换

神经网络中每一层的线性部分通常写成

\[
z = Wx + b
\]

其中 \(W\) 是权重矩阵，\(x\) 是输入向量，\(b\) 是偏置向量。由于这里既有线性变换，也有平移项，因此常称为仿射变换。

!!! abstract "定义 13.1（仿射变换）"
    由线性变换 \(Wx\) 与偏置平移 \(b\) 组成的映射 \(Wx+b\)，称为**仿射变换**。

### 2. 激活函数

激活函数作用在仿射变换输出之后，用于引入非线性。常见的激活函数包括 sigmoid、tanh 和 ReLU。对初学者来说，最重要的不是一开始区分所有激活，而是先理解它们共同在做什么：让模型不再只是纯线性系统。

!!! abstract "定义 13.2（激活函数）"
    作用在线性或仿射变换输出之后、用于引入非线性映射的函数，称为**激活函数**。

### 3. 前向传播

当前向传播发生时，输入经过每一层的仿射变换与激活，逐步流向输出层，最终产生预测结果。这个过程本质上是一个多层复合函数。

!!! abstract "定义 13.3（前向传播）"
    输入从网络前端进入，依次经过各层计算并得到最终输出的过程，称为**前向传播**。

### 4. 反向传播

反向传播是在损失函数计算完成后，利用链式法则把梯度从输出层逐层传回到更前面各层参数上的过程。它的目的不是重新计算前向输出，而是高效得到每个参数应如何更新的方向信息。

!!! abstract "定义 13.4（反向传播）"
    利用链式法则把损失函数对输出的梯度逐层传回各层参数上的过程，称为**反向传播**。

### 5. 计算图

计算图可以把一个复杂模型理解成许多简单运算节点组成的图结构。前向传播就是按图从前到后计算，反向传播则是沿图从后到前传递梯度。这个视角对理解深度学习框架非常有帮助。

!!! abstract "定义 13.5（计算图）"
    把模型中的变量与运算关系表示为节点和边组成的有向图结构，称为**计算图**。

## 例题与推导

### 例 1：单个神经元就是线性打分加激活

设输入向量

\[
x =
\begin{bmatrix}
2 \\
1
\end{bmatrix}
\]

权重向量为

\[
w =
\begin{bmatrix}
0.5 \\
1.0
\end{bmatrix}
\]

偏置为 \(b=-1\)。则线性分数为

\[
z = w^\top x + b = 0.5 \times 2 + 1.0 \times 1 - 1 = 1
\]

若使用 sigmoid 激活，则输出为

\[
\sigma(1) \approx 0.73
\]

这个例子说明，一个神经元最基本的结构其实已经很熟悉：先加权求和，再通过激活转换输出。

### 例 2：为什么没有激活就退化成线性模型

设第一层输出为

\[
h = W_1x + b_1
\]

第二层输出为

\[
y = W_2h + b_2
\]

把第一层代入第二层可得

\[
y = W_2(W_1x + b_1) + b_2 = (W_2W_1)x + (W_2b_1 + b_2)
\]

这仍然只是一个新的线性或仿射变换。这个推导非常重要，因为它清楚说明：如果没有非线性激活，多层堆叠不会真正提升模型表达能力。

### 例 3：链式法则如何出现在网络训练中

设某一层的输出为

\[
h = \sigma(z), \qquad z = wx + b
\]

而损失函数记作 \(L\)。如果想知道 \(L\) 对参数 \(w\) 的梯度，就需要写成

\[
\frac{\partial L}{\partial w}
= \frac{\partial L}{\partial h}
\cdot
\frac{\partial h}{\partial z}
\cdot
\frac{\partial z}{\partial w}
\]

这正是链式法则。也就是说，反向传播并不是一套脱离数学基础的魔法，而只是把第 7 章的链式法则系统化地应用到多层网络里。

### 例 4：ReLU 为什么常用

ReLU 函数定义为

\[
\mathrm{ReLU}(z) = \max(0, z)
\]

如果输入分数为正，它就原样保留；如果为负，则输出 0。这个函数的形式非常简单，但它在实践中常被广泛使用，因为它一方面引入了非线性，另一方面又比某些饱和型激活更容易优化。对初学者来说，不必一开始深入所有实践细节，只要先知道它是在继续承担“引入非线性”的核心任务即可。

## Python 小实验 { #chapter-13-python }

下面这段代码用最小示例展示一个单隐藏层网络的前向计算，以及链式法则如何体现在梯度计算中。重点不是写出完整训练框架，而是让你看到神经网络并没有离开前面的数学基础。

```python
from __future__ import annotations

import math


def relu(value: float) -> float:
    """计算 ReLU 激活值。

    :param value: 输入分数
    :return: 激活后的输出
    """
    return max(0.0, value)


def relu_derivative(value: float) -> float:
    """计算 ReLU 的导数。

    :param value: 输入分数
    :return: 对应位置的导数值
    """
    return 1.0 if value > 0.0 else 0.0


# 单个输入样本与一组简单参数。
x_value: float = 2.0
w1: float = 0.8
b1: float = -0.5
w2: float = 1.2
b2: float = 0.1
target: float = 1.5

# 前向传播：先过隐藏层，再过输出层。
z1: float = w1 * x_value + b1
h1: float = relu(z1)
prediction: float = w2 * h1 + b2

# 用平方误差做一个最小示例损失。
loss: float = (prediction - target) ** 2
print("预测值:", prediction)
print("损失值:", loss)

# 反向传播：链式法则逐层传梯度。
d_loss_d_prediction: float = 2.0 * (prediction - target)
d_loss_d_w2: float = d_loss_d_prediction * h1
d_loss_d_b2: float = d_loss_d_prediction
d_loss_d_h1: float = d_loss_d_prediction * w2
d_loss_d_z1: float = d_loss_d_h1 * relu_derivative(z1)
d_loss_d_w1: float = d_loss_d_z1 * x_value
d_loss_d_b1: float = d_loss_d_z1

print("对 w2 的梯度:", d_loss_d_w2)
print("对 b2 的梯度:", d_loss_d_b2)
print("对 w1 的梯度:", d_loss_d_w1)
print("对 b1 的梯度:", d_loss_d_b1)
```

如果你运行这段代码，会发现前向部分只是若干次很熟悉的数值计算，而反向部分则完全是在做链式法则的逐层展开。这正是本章最希望建立的直觉：神经网络训练并不是黑箱，而是前面数学基础的大规模组合。

## 与机器学习的联系

### 1. 神经网络是在重复使用线性模型的核心结构

每一层的核心仍然是 \(Wx+b\)。因此，神经网络并不是脱离线性模型的对象，而是在线性组合的基础上加了层次和非线性。

### 2. 激活函数决定模型能否表达复杂关系

如果没有非线性激活，多层网络就会退化成单层线性结构。也正因为如此，激活函数不是局部小技巧，而是网络表达能力的关键来源。

### 3. 反向传播把链式法则真正变成训练算法

前面第 7 章中的链式法则在本章第一次以完整训练机制的形式出现。理解这一点之后，自动求导和深度学习框架的很多行为都会更容易理解。

## 常见误区

### 误区 1：神经网络是一种完全不同于前面数学的黑箱

神经网络看起来复杂，但底层仍然建立在向量、矩阵、函数、导数和优化之上。它的复杂来自组合规模，而不是数学基础被推翻。

### 误区 2：层数越多就一定越好

更深的结构意味着更强表达能力，但也意味着更难训练、更易出现优化和泛化问题。深度不是单向度的“越多越高级”。

### 误区 3：激活函数只是为了让公式更复杂

激活函数的根本任务是提供非线性。如果没有它，多层线性堆叠不会真正提升模型表达能力。

### 误区 4：反向传播是一套只能死记的专有算法

反向传播的本质就是链式法则在多层结构中的系统应用。理解这层数学来源，比机械背步骤更重要。

## 本章小结 { #chapter-13-summary }

本章最核心的任务，是帮助读者看到神经网络并没有跳出前面学过的数学，而是在系统地重复与组合这些基础对象。单个神经元先做仿射变换 \(Wx+b\)，再通过激活函数引入非线性；多层网络则是在逐层构造更适合任务的表示；而训练中的反向传播，则把链式法则真正变成了逐层传递梯度的算法机制。只要这三层结构真正建立起来，神经网络就不会再显得像神秘黑箱。

从整本书的角度看，本章也是一次重要综合：向量、矩阵、函数、导数、优化和概率解释，都会在神经网络中重新出现。对第一次学习的人来说，本章真正需要带走的，不是某一类网络的繁琐实现，而是一个稳定认识：深度学习的复杂外观之下，仍然是前面学过的数学在更大规模上的协调运作。

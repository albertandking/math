1. 用 uv 创建和管理 Python 虚拟环境，uv 按照库，则需要使用 pypi 官方镜像
2. 本项目为面向零/弱数学基础学生的中文机器学习数学教材，正文默认使用中文书写
3. 教材正文采用“中文教材段落体”，以连续段落讲述为主，避免一句一行的碎片化表达
4. 列表只用于并列信息、步骤、小结、练习等场景，不能替代正文讲解
5. 每章需包含明确的学习目标、读者预期、本章动机，并与机器学习场景建立联系
6. 对抽象概念优先采用“绘图 + 举例 + 段落讲解”的方式，再进入公式定义
7. 数学公式统一使用 LaTeX 书写，行内用 `\(...\)`，独立公式用 `\[...\]`
8. 正文中的 Python 代码示例必须带注释，且代码前后应有解释性文字，不得只堆放代码块
9. 图、公式、代码都必须嵌入正文叙述中，服务于讲解，不应孤立罗列
10. 已成稿章节优先做教材化文体统一，再继续扩写后续新章节
